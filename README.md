# Inprodi

1. Clonar proyecto
2. ```npm i```
3. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```
4. Cambiar las variables de entorno
5. Levantar la base de datos
```
docker-compose up -d
```

6. Levantar: ```npm run dev```

7. Ejecutar SEED 
```
http://localhost:8080/api/seed
```


## Stack Usado
- TypeOrm Mysql
- NestJs
